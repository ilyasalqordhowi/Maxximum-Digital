#GETTING STARTED

1. Clone the repository:

   ```bash
   git clone https://github.com/ilyasalqordhowi/Maxximum-Digital.git
   cd <project-name>
   ```

2. Run the program in VSCode:
   ```sh
   code .
   ```
3. go to file frontend:
    ```sh
   cd frontend
   ```

4. Install dependencies:
    ```sh
   npm install
   ```
    
5. Run the program:

```sh
  npm run dev
```
